# Smart E-Learning

Dự án nền tảng học trực tuyến thông minh sử dụng React, Node.js và Google Cloud.

## Công nghệ sử dụng

- Frontend: React + Vite
- Backend: Node.js + Express
- Database: Firestore
- Storage: Google Cloud Storage
- Authentication: Firebase Authentication
- AI: Vertex AI / Gemini API
- Deployment: Cloud Run
